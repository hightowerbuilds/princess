# Example Landing Page Beats

Use these as examples, not mandatory copy.

## Hero

Princess is a local inbox for prompts that are too useful to leave in chat.

## Workflow

1. Ask an agent to draft or refine a prompt.
2. The agent saves it to Princess.
3. Open the TUI later to browse, edit, copy, or package it.

## Proof Points

- Files stay readable on disk.
- Markdown works for simple prompts.
- HTML workspaces work for structured prompts with sources and assets.
- Agents can use the same CLI humans use.
