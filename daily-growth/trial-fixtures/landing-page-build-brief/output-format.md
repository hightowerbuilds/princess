# Expected Output Format

Return a build-ready landing page plan with:

- **Page Structure:** ordered sections with purpose and content notes.
- **Hero Copy:** headline, supporting copy, and primary action.
- **Component Notes:** reusable UI components and states.
- **Responsive Behavior:** mobile, tablet, and desktop layout guidance.
- **Content Risks:** claims or wording that need product confirmation.
- **Implementation Checklist:** concrete steps for the next coding agent.
